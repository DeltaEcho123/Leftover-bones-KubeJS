# Leftover-bones-KubeJS
The official leftover bones mod port to fabric and quilt

This is the official port of the mod leftover bones to fabric and quilt. It uses the mod KubeJS as "mod loader"
This mod will load on any mod loader with a compatible KubeJS installation.
This mod has only been tested on Minecraft 1.20.1 
Newer versions should work but is not guaranteed.
It is not recommended to use this mod with versions older than 1.20.1

If you are playing on forge or neoforge, it is highly recommended you download the mod from curseforge. [https://www.curseforge.com/minecraft/mc-mods/leftover-bones](url)
