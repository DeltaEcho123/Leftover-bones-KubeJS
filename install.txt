1. Make sure you have KubeJS installed.
2. Enter the folder "kubejs" inside your game files. It is automatically generated after running the game for the first time0
3. Move the files/folder:
  - "server_script_leftoverbones - 1.0.0.js" into "server_scripts"
  - "startup_script_leftoverbones - 1.0.0.js" into "startup_scripts"
  - "leftoverbones" into "assets"
4. Run Minecraft and enjoy! 
Note: This mod has only been tested on Minecraft 1.20.1 Newer versions should work but is not guaranteed.
